What is CLOUD

Compared to the USB or mobile hard disk drive, people need a new method to carry or save their files. At the same time, this new method needs to be safety, and response quickly. That is the main purpose of the cloud storage, but cloud storage only is a part of cloud computing, there are serval different way to use Cloud such as cloud game, big data, or cloud education.

The target audiences of this website are those people who have none or little understanding about cloud technology. This website will explain to audiences the difference of each cloud service, also introduce the basic information and usages. Which will help audiences understand more about cloud technology and what can they do with that.

Cloud era

Cloud game

A magazine called Cloud gaming: architecture and performance defined the cloud gaming is a online gaming technology based on cloud computing, under this technology, computers with limited graphics processing and data computing capabilities to run high-quality games. In cloud gaming, players will process the game in the cloud server instead their own computer, and the cloud server renders the scene into video and audio stream and transfer it to the player’s computer or other game devices through internet. This technology means that player’s devices do not need strong graphics processing and data computing capabilities any more, they only need to have basic streaming media capabilities, ability to get player connect to the cloud server and transfer commands.

Big data

In the past, big data can not be processed by traditional database system because the cost are too expensive, those system can not save unstructured data such as pictures, texts or videos, and they can not resize themselves to fit that large amount of data. google cloud we can see that with the development of internet, companies and organizations will have massive amounts of unstructured data through online customer interaction every day. In addition of this, internet of things such as phone, computer, pad, also make the amount of data like pictures, text or videos have an explosive growth trend. Compared to the local deployment, cloud technology can provide a safety, flexible platform with low cost, customer can save or process their data safely when they need to and get value from relevant data.

Cloud computing

Cloud computing is a type of distributed computing, which means that processing programs with huge data into small programs through the ‘cloud server’, and analyse those programmes through a system composed of multiple servers, then return the result to the user. Through cloud computing, users can use all the resources online depends on their demand, and can be seen as infinitely expanding(Google Cloud), You can also think cloud as a waterworks, you can get unlimited wate when you need, you only need to pay for what you got.
